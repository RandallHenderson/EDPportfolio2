Robotics Project

Yahboom Micro:Bit Robot

Fully programming robot with many sensors and capabities!

Project Introduction

The purpose of this project was to implement the Hummingbird Robotics Kit into a unit or lesson. Again, my goal was to make something meaningful and not a novelty. I remembered an activity I saw a few years ago at the Creighton University Physics Field Day that required students to guide a laser through a maze using various mirrors and lenses. The catch was students couldn't have the laser ON while they set everything up. I decided to adapt this project to my physics class while teaching Law of Reflection. The Hummingbird ended up being an interactive target that served as the "finish line" for the laser maze.

This is the only project that I was able to use in my own class this semester, so you will actually see some student work on this page.

My project was no different. If you can get your hands on the Hummingbird Kit that suits your needs, challenge yourself to make something with the materials you already have in your house! Hummingbird Details
Plugging sensors into the Hummingbird is very straightforward. Each sensor or output device has a specific, clearly labeled location to attach. The picture below shows my Hummingbird with all of my "parts" plugged in. This was my first attempt at my original idea which included wheels, hence the DC motors. The final version ditched the wheels in favor of vibrating motors (not included in the kit linked in the materials section, but not necessary). The other components attached are 2 RGB LEDs, 1 Light Sensor, and 1 Servo.

  • Programming
  • There are several different ways to program the Hummingbird based on your level of expertise and comfort with varying programming languages. I chose to use the "Snap!" programming environment. The nice think about Snap! is that it is very simple and takes very little time to learn how to use. There are even special commands build into the environment specifically for using the Hummingbird Kit. At the bottom of the page, you will find a link to download code that I used, or feel free to experiment with your own input and output settings! My particular project waits with the flag down and LEDs illuminated green until the the light sensor detects the laser. When the laser hit, it will wave the flag and blink the LEDs red ten times, then pause and say "The building has been hit" through the computer speakers. My students had fun changing the computer to say different phrases for each group that hit the target! Below is a screenshot of how all my "blocks" fit together to accomplish, or you can download the XML at the bottom of the page and import it on the Snap! website.

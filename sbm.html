Project Introduction

This project made use of the Particle Photon. The Photon is a microcontroller similar to the Arduino platform. The neat thing about the Photon is that it is a web-based platform. All programming is done via Particle's online development environment and the board itself is easy to incorporate with web servers and popular websites. My personal challenge for this project was to develop a tool that would use this function for a solid educational purpose rather than the novelty of having internet. What I came up with was a twist on a heat transfer/insulation project I already do with my physics students. The students have the goal of restricting the flow of heat from one side of a cooler to another. A heat lamp is placed in one chamber, a thermometer in the other, and the students build the insulating wall that separates them. Previous versions of the project involved thermometers or digital temperature probes where students would record data manually and graph their results. The Photon eliminates the tedious work of manual record keeping and instead sends the data automatically to a Google Sheets spreadsheet. From here, students can manipulate and interpret their data much easier than with previous methods. Using the Google platform also allows the data to be easily shared between group members or the teacher.
